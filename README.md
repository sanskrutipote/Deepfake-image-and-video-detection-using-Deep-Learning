# Deepfake-image-and-video-detection-using-Deep-Learning
A real-time deepfake detection system leveraging EfficientNetB4 with integrated attention and Siamese learning. Detects and classifies manipulated faces in videos, including Deepfakes, FaceSwap, Face2Face, and Neural Textures, delivering accuracy and model interpretability.
